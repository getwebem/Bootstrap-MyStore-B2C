# MyStore E-commerce website
### **Getting started**
### Technologies Used
#### HTML5, CSS3, Bootstrap 3.3, CSS Preprocessor/LESS
 - Run :  `git clone https://github.com/getwebem/myStore.git`
 - Run :  `cd myStore`
 - Run :  `Brackets .`
 - Open:  `Index.html`
 - Open:  `Live Preview`
